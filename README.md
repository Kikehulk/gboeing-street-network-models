# Urban Street Network Models and Indicators

This project generates street network models and indicators for every urban area in the world and shares them in an open data repository.

**Citation info**: Boeing, G. 2020. [Street Network Models and Indicators for Every Urban Area in the World](https://doi.org/10.31235/osf.io/f2dqc). doi:10.31235/osf.io/f2dqc

## Data repository

All the models and indicators are shared in an open [data repository](https://dataverse.harvard.edu/dataverse/global-urban-street-networks/) on the Harvard Dataverse.
